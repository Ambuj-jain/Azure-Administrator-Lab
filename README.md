# Azure Administrator Lab

## Overview
This repository contains hands-on Microsoft Azure administration labs designed to demonstrate practical cloud administration skills aligned with AZ-104 certification objectives.

## Objectives

- Deploy and manage Azure Virtual Machines
- Configure Virtual Networks
- Manage Storage Accounts
- Implement Azure Backup
- Configure Monitoring and Alerts
- Manage Identity and Access

## Technologies

- Microsoft Azure
- Azure Virtual Machines
- Azure Storage
- Azure Backup
- Azure Monitor
- Virtual Network
- Network Security Groups

## Lab Modules

### Module 1 - Resource Groups
- Create Resource Groups
- Manage Resources

### Module 2 - Virtual Machines
- Create Windows VM
- Configure RDP Access
- VM Monitoring

### Module 3 - Networking
- Virtual Networks
- Subnets
- NSG Configuration

### Module 4 - Storage
- Storage Accounts
- File Shares
- Blob Storage

### Module 5 - Backup & Recovery
- Recovery Services Vault
- VM Backup
- Restore Operations
